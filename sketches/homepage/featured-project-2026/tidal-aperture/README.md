## Variant: Aperture / 潮汐窗口

### Design stance

把 Featured Project 做成一扇沉浸式窗口，项目视觉成为主叙事，文字退到边缘。

### Key choices

- Layout: 深色切角视觉场 + 左侧窄信息轴
- Typography: 巨幅标题跨越信息区与视觉区
- Interaction: 指针光场、切角扩张、潮汐层缓慢错位

### Trade-offs

- Strong at: 冲击力、沉浸感、首屏记忆点
- Weak at: 与当前浅色首页反差最大，需要精心处理上下文过渡

### Best for

- 想让心屿成为整个首页最强的品牌时刻
