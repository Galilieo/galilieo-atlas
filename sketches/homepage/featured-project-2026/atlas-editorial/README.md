## Variant: Atlas / 编辑地图

### Design stance

把项目卡变成一张可探索的项目地图，让品牌线稿、标题与元数据共享同一版面。

### Key choices

- Layout: 非对称海报网格，岛屿等高线跨越版面
- Typography: 大尺度编辑衬线 + 微型 mono 坐标
- Interaction: World / System 双视图，轮廓轻微位移与相位变化

### Trade-offs

- Strong at: 编辑感、品牌识别、与现有整站风格兼容
- Weak at: 对移动端排版细节要求最高

### Best for

- 想保留克制浅色基调，同时明显摆脱普通项目卡
