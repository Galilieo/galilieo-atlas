## Variant: 编辑式选择面板

### Design stance

把所有允许项直接展示在设置抽屉里，让分类、标签和封面选择无需二次弹层，移动博客保持舒展的单列编辑感。

### Key choices

- Studio：分类为单选卡；标签为只过滤已有项的搜索 + 多选 chips；封面为自动匹配、图库缩略图和专属上传。
- 移动博客：单列、164px 图片、21px 两行标题、13px 两行摘要、18px 内容内边距。
- 交互：分类单选、标签切换、标签过滤、封面切换、保存 toast、Studio/博客视图切换。

### Trade-offs

- Strong at：长中文标题可读；选择状态一眼可见；学习成本低。
- Weak at：设置抽屉更长；一屏只能看到约一篇博客卡。

### Best for

- 日常写作频率不高、希望“打开就知道有哪些可选项”的个人作者。
- 保留 Galilieo 当前 Editorial Atlas 单列移动阅读感。
